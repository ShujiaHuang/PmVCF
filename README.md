### addformat

	Usage : perl vcfutil.pl addformat -from [vcf] -to [Target vcf] -add 'foo:bar' > OutputVCF

	CAUTION The samples and the order of sample should be the same in [-from [vcf]] and [-to [Target vcf] ]
